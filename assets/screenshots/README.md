# DeskLab Store — Screenshots

This folder is reserved for public screenshots used in the README, portfolio pages and release notes.

Recommended screenshots:

```text
homepage.png
catalogue.png
mobile_homepage.png
mobile_catalogue.png
```

Recommended capture flow:

1. Open the live site:
   `https://antonmorosi06.github.io/desklab-store/`

2. Capture the top homepage section.

3. Scroll to the product catalogue.

4. Capture the product catalogue section.

5. Optionally capture mobile layout using browser responsive mode.

The screenshots should show the actual live site, not a fake mockup.
