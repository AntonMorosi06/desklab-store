## Main improvements

The v0.3 release establishes DeskLab Store as a more serious and documented project. The release includes a cleaner storefront, premium motion files, a structured JSON catalogue and documentation that clearly separates implemented features from planned or manual ones.

## Technical highlights

- Static frontend using HTML, CSS and JavaScript.
- Product catalogue stored in `products.json`.
- Premium visual/motion layer through `v04-premium.css` and `v04-premium.js`.
- Lightweight SVG assets.
- GitHub Pages deployment.
- Documentation under `docs/`.
- Draft legal notes under `legal/`.
- Supplier and margin files under `data/`.

## Known limitations

This release does not include:

- backend
- database
- payment gateway
- automated checkout
- inventory tracking
- order dashboard
- customer accounts
- automated emails
- final legal policy

## Release meaning

This release should be treated as a validation baseline, not as a production store. Its value is in the combination of static frontend, product catalogue structure, public deployment and honest roadmap.
